ü§î What is this all about?

Imagine you're a detective looking for clues. You have a bunch of data points, like how many hours your friends study and the test scores they get. Linear regression is your special detective tool for finding a hidden pattern!

It helps you draw a single straight line that cuts right through the middle of all your data points. This is called the "line of best fit." Once you have this magic line, you can make predictions!

This special line has two important parts:

  • Slope: This is the 'steepness' of the line. It tells you how much your prediction goes up (or down) for every step you take. For example, a slope of 8 means "for every 1 extra hour you study, your score is predicted to go up by 8 points."
  • Intercept: This is the 'starting point.' It's the value where the line hits the vertical axis (when the horizontal value is zero). For example, an intercept of 42 means "if you studied for 0 hours, your predicted score would be 42."

üêç The Python Code

Here is the Python code to perform the linear regression we talked about. You can run this in an environment like Google Colab.

# Step 1: Import the tools we need
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# Step 2: Create our dataset
# x-axis: Hours Studied
study_hours = np.array([2.5, 5.1, 3.2, 1.5, 4.8, 6.2, 0.5, 3.8])

# y-axis: Exam Score
exam_scores = np.array([65, 82, 70, 50, 88, 92, 45, 78])

# Step 3: Use SciPy to find the magic line of best fit! üßô‚Äç‚ôÇÔ∏è
slope, intercept, r, p, std_err = stats.linregress(study_hours, exam_scores)

# Let's see what we got!
print(f"üìà The slope is: {slope:.2f}")
print(f"üéØ The intercept is: {intercept:.2f}")

# Step 4: Make a prediction!
# What's the predicted score for 4.5 hours of study?
# Use the formula: y = mx + c
hours_to_predict = 4.5
predicted_score = slope * hours_to_predict + intercept
print(f"\nüîÆ Predicted score for {hours_to_predict} hours: {predicted_score:.2f}")

# Step 5: Draw a picture to see our line!
def my_regression_line(x):
  return slope * x + intercept

line_values = my_regression_line(study_hours)

plt.scatter(study_hours, exam_scores, label='Original Data Points')
plt.plot(study_hours, line_values, color='red', label='Line of Best Fit')
plt.scatter(hours_to_predict, predicted_score, color='green', s=100, zorder=5, label=f'Prediction for {hours_to_predict} hrs')

plt.title('Study Hours vs. Exam Score')
plt.xlabel('Hours Studied')
plt.ylabel('Exam Score')
plt.legend()
plt.grid(True)
plt.show()
