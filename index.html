Finding the Extremes: max() and min() üìàüìâ

Imagine you have a list of high scores from a game. You want to know the absolute best score and the lowest score. NumPy makes this a breeze!

  • .max(): Finds the largest value in your array.
  • .min(): Finds the smallest value in your array.

Example Code:

import numpy as np

game_scores = np.array([120, 85, 205, 150, 99])

highest_score = game_scores.max()
lowest_score = game_scores.min()

print("Highest Score:", highest_score) # Output: Highest Score: 205
print("Lowest Score:", lowest_score)   # Output: Lowest Score: 85

Calculating the Average: mean() üìä

When you want to know the "typical" value in a list of numbers, you calculate the average. In math class, you add all the numbers and divide by how many there are. NumPy's mean() function does this for you instantly!

The formula for the mean ($\mu$) is:

$$ \mu = \frac{\sum_{i=1}^{N} x_i}{N} $$

Where:

  • $ \sum x_i $ means "add all the numbers"
  • $ N $ is the count of numbers

Example Code:

import numpy as np

student_heights = np.array([150, 155, 160, 148, 152]) # Heights in cm

average_height = student_heights.mean()

print("Average Height:", average_height) # Output: Average Height: 153.0

Summing It All Up: sum() ‚ûï

Sometimes, you just need to add all the numbers in your array together. Maybe it's the total points from a game level, or the total amount of money collected. The sum() function is perfect for this.

Example Code:

import numpy as np

daily_sales = np.array([100, 150, 75, 200, 125]) # Sales for 5 days

total_sales = daily_sales.sum()

print("Total Sales for the Week:", total_sales) # Output: Total Sales for the Week: 650

How Many Numbers? size üì¶

It's not a function (it doesn't have parentheses ()), but a very useful attribute (like a property) of a NumPy array. .size tells you how many elements (numbers) are in your array. This is super helpful for knowing how much data you're working with.

Example Code:

import numpy as np

my_shopping_list_items = np.array(['milk', 'eggs', 'bread', 'juice', 'cheese']) # This array contains strings, but .size works similarly for numbers

number_of_items = my_shopping_list_items.size

print("Number of items on my list:", number_of_items) # Output: Number of items on my list: 5
