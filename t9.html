Principle 1: Write Clear and Specific Instructions

Use the following tactics to make your instructions to an AI unambiguous and more likely to produce the desired output.

Concept: Use delimiters to clearly indicate distinct parts of the input. Delimiters can be anything like: ```, """, < >, <tag> </tag>, etc.

Example (Python f-string): This shows how to structure a prompt that separates instructions from the text to be processed.

text = f"""
You should express what you want a model to do by 
providing instructions that are as clear and 
specific as you can possibly make them. 
This will guide the model towards the desired output, 
and reduce the chances of receiving irrelevant 
or incorrect responses. Don't confuse writing a 
clear prompt with writing a short prompt. 
In many cases, longer prompts provide more clarity 
and context for the model, which can lead to 
more detailed and relevant outputs.
"""

prompt = f"""
Summarize the text delimited by triple backticks 
into a single sentence.
```{text}```
"""

Concept: Request the output in a specific format like JSON or HTML. This makes the response predictable and easier to use in other programs or workflows.

prompt = f"""

Assignment: Mastering the Art of the Prompt ‚úçÔ∏è

Your goal is to write four distinct prompts, each one demonstrating your understanding of a key tactic for clear and effective communication with an AI.

1. The Delimiter Challenge

Find a short paragraph of text from any source (a news article, a book, a website). Your task is to write a prompt that asks an AI to perform an action on this text.

  • Requirement: You must use delimiters (like """...""" or ---...---) to clearly separate your instructions from the text you want the AI to analyze.
  • Example Actions: Summarize the text, translate it, extract all the names of people, or change its tone from formal to casual.

2. The Structured Output Challenge

Imagine you are building an application that needs data about a specific topic (e.g., planets, famous artists, types of cheese, car manufacturers).

  • Requirement: Write a prompt that asks the AI to generate at least three examples of your chosen topic. You must specify that the output should be in JSON format and define the exact keys you want to see in the response (e.g., name, discovery_date, mass).

3. The Conditional Challenge

Write a prompt that instructs an AI to analyze a piece of text and perform a different action based on what it finds.

  • Requirement: Your prompt must include a clear condition. For example, "If the following customer review is positive, write a cheerful thank you message. If it is negative, write a polite apology and ask for more details."

4. The Style-Mimic Challenge (Few-shot)

Your task is to teach an AI a new, specific style of responding by giving it an example to follow.

  • Requirement: Write a single prompt that includes at least one complete example of an input and the desired output style. Then, provide a new input and ask the AI to apply that same style.
  • Example Ideas: Teach it to turn movie titles into limericks, describe historical events like a pirate, or answer questions with another question.
