They want to train a computer program to automatically recognize and name the character in any new drawing they create. Is this an example of Supervised Learning?

They want to build a model that can predict a new player's final score at the end of their first month. They use the historical data, which includes the final scores of all past players, to train the model. Is this an example of Supervised Learning?

Does the data used for training already contain the exact thing the program is trying to predict?

They want a program to sort these photos into different groups based on visual similarities (e.g., animals with feathers, animals with fur). The program is not told beforehand what species exist or how many groups to create. Is this an example of Supervised Learning?

Is the computer being asked to categorize photos into pre-defined labels, or is it creating the categories itself?

Focus on whether the algorithm is discovering groups of users on its own or learning from a list of 'perfect' song recommendations.
