1. The Core Concept: Super Autocomplete

Before meeting the Titans, explain the "Super Autocomplete" analogy:

  • The Phone: Guesses the next word based on the last 3 texts you sent.
  • An LLM: Guesses the next word based on almost everything ever written on the internet.
  • The Probability Game: AIs assign "points" to words. In "The dog jumped over the...", Fence might get 99 points, while Moon gets 1 point.

Tokens: The DNA of Memory

AI doesn't count words; it counts Tokens (word chunks).

The Activity: Ask ChatGPT and Llama: "How many tokens are in the sentence: 'The prehistoric pterodactyl was incredibly aerodynamic'?"

The Lesson: Simple words are 1 brick; complex words are 3 bricks. If you run out of tokens, the AI's "memory" is full.

The "Desk" Limit (Context Window)

Every LLM has a "Desk Size." Once full, old info "falls off" and is forgotten.

The Activity: Give an AI a secret: "My favorite color is Magenta." Then paste 10 pages of a book. Ask: "What is my favorite color?"

The Lesson: Larger windows = bigger desks. Some models remember better than others.

Lost in the Middle (The Haystack)

AIs get "distracted" by stuff in the middle of a long conversation.

The Activity: Paste a massive Wikipedia article. Hide the sentence: "The secret code is: PINEAPPLE" right in the middle. Ask the AI for the code.

The Lesson: Important info should go at the start or the end of your prompt, not the middle.

RAG: The External Hard Drive

When the desk is too small, AI uses Retrieval-Augmented Generation (RAG) to look things up.

The Activity: Ask Llama (offline) and Gemini (connected): "What was the weather in Paris yesterday?"

The Lesson: RAG is like a library card; it helps the AI learn things it wasn't "born" knowing.

4. The "Vibe Check" Coding Challenge

Test the personality differences between Claude and ChatGPT.

The Prompt: "Write a Python script using the turtle library to create a 'Hypnotic Space Spiral.' The background should be black, the pen should cycle through neon colors, and it should draw very fast."
Look for: Claude will be precise and give clean code. ChatGPT will talk a lot more and act like a "Supportive Coach."
